Модельно-методический аппарат

Модельно-методический аппарат представлен несколькими взаимосвязанными математическими моделями. После выполнения расчётов происходит уточнение значения погрешности. При необходимости процесс зацикливается для увеличения точности результата.

Программа для вычисления параметров самолёта реализована с использованием языка программирования Python версии 3.7.3. Она состоит из двух файлов: папки с базой данных и файла типа 'exe' .
